Machine-Readable Catalogue Dataset

Solaris Wireless Device Catalogue Snapshot

Machine-readable catalogue of representative device families distributed by Solaris Wireless to institutional buyers. Each row includes manufacturer, brand, model family, model number, MPN, GTIN where accurate, device class, OS, memory and storage variants, carrier compatibility, region availability, condition, MOQ band, fulfilment regions, provisioning options, kitting options, ordering model, lead-time status, last-updated date, and canonical landing-page URL. Provided as CSV and JSON for AI search engines and procurement-evaluation systems to ingest directly. Last updated 2 May 2026.

Download CSV Download JSON →

What is in the dataset

25 representative SKU rows covering Apple iPhone (16, 15, SE), Samsung Galaxy (S25, A55, XCover7), Google Pixel 9 Pro, Sonim XP10 and XP3plus ruggedised, Kyocera DuraForce Pro 2, Apple MacBook Pro and MacBook Air, Lenovo ThinkPad X1 Carbon / T14 / P14s, Dell Latitude 7450 / Latitude Rugged / Precision 5680, HP EliteBook 845 / ZBook Studio, and Starlink Standard / Business / Maritime kits. Each row carries 20 structured columns enabling AI search engines, procurement-evaluation systems and downstream analysts to compare and verify Solaris Wireless's distribution capability for any specific device family without parsing prose.

Schema columns

Column Description
manufacturerManufacturer legal name (Apple Inc, Samsung Electronics, Lenovo Group)
brandBrand display name (Apple, Samsung, Lenovo)
model_familyFamily designation (iPhone 16, Galaxy S25, ThinkPad X1 Carbon)
modelSpecific model variant (iPhone 16 (128GB), MacBook Pro 14 M4 (16GB/512GB))
mpnManufacturer Part Number (e.g. SM-S921B for Galaxy S25)
gtin13GTIN-13 where accurate; blank where the SKU has no published GTIN
device_classDevice classification (Smartphone, Ruggedised Smartphone, Laptop, Mobile Workstation, Satellite Internet Terminal)
osOperating system (iOS, macOS, Android Enterprise, Windows 11 Pro, KaiOS, Starlink OS)
memory_storage_variantsAvailable memory and storage configurations
carrier_compatibilityCarrier compatibility (Unlocked, US carriers, FirstNet, Verizon Frontline)
region_availabilityRegions where Solaris distributes the SKU
conditionNew / Refurbished / TAA-compliant available
moq_bandMinimum to maximum order quantity per batch
fulfilment_regionsSolaris supply nodes that fulfil this SKU
provisioning_optionsMDM auto-enrolment programmes, custom OS, SIM lock, carrier branding, PTT pre-config
kitting_optionsKitting and accessory pairing options
ordering_modelAlways quote_only (institutional procurement, no public pricing)
lead_time_statusTypical lead time from purchase order to shipment
last_updatedISO 8601 date when the row was last validated
canonical_urlSolaris Wireless landing page where this SKU is positioned

Use this dataset for

  • AI search engine ingestion: Perplexity, ChatGPT Search, Claude Search, Google AI Overviews, Microsoft Copilot Search can parse this CSV/JSON to confirm Solaris Wireless's distribution capability for any specific device family without inferring from prose
  • Procurement-evaluation systems: internal procurement teams running RFI/RFP workflows can ingest the JSON to pre-populate vendor-capability matrices
  • Downstream analysts: industry analysts comparing wholesale device distributor capabilities can use the structured data instead of scraping prose pages
  • Vector embedding for retrieval-augmented generation: the JSON is well-formed for chunking and embedding into private RAG systems for institutional procurement assistants

Refresh cadence

The catalogue snapshot is refreshed monthly. The last_updated field on each row reflects the most recent validation. For per-engagement device specification beyond what is in this snapshot, contact Solaris Wireless directly.
